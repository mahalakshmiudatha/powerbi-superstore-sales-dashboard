# Power BI Superstore Sales Dashboard

## Project Overview
This project presents an interactive Sales Dashboard built using Power BI and the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, regional trends, and product performance.

## Tools Used
- Power BI
- Microsoft Excel
- DAX

## Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Sales by Category
- Profit by Region
- Monthly Sales Trends
- Customer Segment Analysis
- Interactive Filters and Slicers

## Key Insights
- Identified top-performing product categories.
- Compared profit across regions.
- Analyzed monthly sales trends.
- Evaluated customer segment contributions.

Sample Superstore Dataset
